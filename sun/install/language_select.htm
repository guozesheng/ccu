请选择安装PPFrame的语言版本

GBK：国内用户推荐使用GBK编码，PPFrame从一开始就做好了应用程序国际化的规划，没有丝毫语言障碍。盲目崇拜UTF-8是错误的。

UTF-8：国外用户、繁体用户请选择UTF-8。

BIG5及其他：理论上，PPFrame只需要在模板目录下新建语言目录，然后将GBK目录内的文件拷贝过去，并且转码为对应目录即可迅速支持任何语言版本。由于受众有限，目前不提供。

安装 GBK版本（推荐） 这是官方开发版本，总是最新的！

安装UTF-8 版本 UTF-8版本模板不一定是最新的，如非必须，请用GBK。

语言版本的选择只在首次安装时有意义，升级时没有任何意义。

版本号说明 更新日志

  • PPFrame有一个版本号，这是基础框架核心版本号，在后台首页可以看到
  • 各个模块有自己的版本号，在各个模块后台首页可以看到
  • 整个压缩包的版本号是PPFrame版本号与各模块版本号之中的最大值
  • 版本号前三节为具体版本号，第四节为更新日期6位数字，记录PPExam的历史
  • 版本号是系统诊断与升级的唯一依据，请勿更改系统版本号
